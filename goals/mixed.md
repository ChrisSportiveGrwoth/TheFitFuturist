## How to use this file

These are principles for concurrent training. Load both `runner.md` and `strength.md` alongside this file — the principles in both apply. This file adds the interference management layer on top.

---

## Principle 1: The Interference Effect — What It Actually Is

**The principle:**
Combining strength and endurance training in the same program can reduce gains in both compared to training only one quality. This is the "interference effect."

**The nuanced reality (based on current evidence):**
- The interference effect is real — but less severe than originally thought
- It primarily affects lower-body strength and power adaptations
- Upper-body strength is largely unaffected by concurrent endurance training
- In untrained individuals and women: interference is minimal — both qualities improve simultaneously
- In trained individuals: interference becomes more significant
- Running causes more interference than cycling (due to greater muscle damage)

**Implication for planning:**
Do not frame concurrent training as a problem to eliminate — frame it as a constraint to manage intelligently. The goal is to minimize molecular signal conflict between adaptations, primarily AMPK (endurance signal) vs. mTOR (strength signal).

## Principle 2: Sequencing — Within Session and Between Sessions

**Within the same session (when unavoidable):**
- Strength before endurance: better for strength adaptations
- Endurance before strength: better for endurance performance (prioritize if endurance is primary goal)
- Evidence: sequence effect on strength adaptations is more significant; sequence effect on endurance performance is moderate — expect meaningfully reduced quality (up to 8-12% reduction in peak power for intervals) if endurance follows heavy compound strength within 6h

**Between sessions (the primary lever):**
- Separate strength and endurance sessions by 6+ hours minimum
- Ideally: different days entirely
- High-intensity endurance before leg strength: avoid (pre-fatigues CNS and muscles)
- Leg strength before high-intensity endurance: avoid (residual fatigue reduces endurance quality)

**The muscle-group matching rule (key principle):**
The interference effect is greatest when the same muscle groups are used in both sessions. Therefore:

| Combination | Interference | Rule |
|---|---|---|
| Leg strength + running | HIGH | Separate by 48h or more |
| Leg strength + cycling | MODERATE | Separate by 24-48h |
| Upper body strength + running | LOW | Same day acceptable |
| Upper body strength + cycling | LOW | Same day acceptable |
| Leg strength + easy run (Z1) | LOW | 6-8h gap acceptable |
| Leg strength + intervals/tempo | HIGH | Never same day, 48h minimum |

## Principle 3: Priority Hierarchy

**The principle:**
Mixed athletes must prioritize. "Balanced" training without a priority leads to mediocre outcomes in both areas. Identify the user's primary goal and build around it.

**Derive priority from assessment:**
- If running performance is primary → load `runner.md` as primary, use `strength.md` for supplementary work
- If strength/muscle is primary → load `strength.md` as primary, use `runner.md` principles for cardio programming
- If truly balanced → use block periodization (see Principle 4)

**Priority affects volume allocation:**
- Primary modality: full volume prescription from the relevant goal file
- Secondary modality: reduced volume (50-70% of standalone prescription)
- For strength as secondary: frequency reduction matters more than set count — 2 strength sessions/week is the evidence-based minimum effective dose for concurrent runners. A 3rd strength session adds interference risk disproportionate to the strength benefit.
- For endurance as secondary: reduce intensity before volume — keep easy runs easy (Z1-2), cut interval frequency first when adjusting down

## Principle 4: Periodization for Mixed Training

**The principle:**
Concurrent athletes accumulate fatigue faster than single-sport athletes. Deloads are more critical.

**Block periodization (for balanced athletes):**
Alternate emphasis every 4-6 weeks rather than trying to develop both qualities simultaneously at maximum intensity:
- Block A: Emphasize endurance (higher run volume, maintenance strength)
- Block B: Emphasize strength (higher lifting volume, maintenance cardio)

This reduces chronic interference while building both qualities over time.

**Intensity distribution for concurrent athletes:**
Concurrent athletes should avoid heavy Z3 (threshold) accumulation — Z3 work generates disproportionate fatigue that interferes with both strength adaptations and high-quality endurance sessions. Use a polarized approach:
- ~80% of endurance sessions at Z1-2 (easy, conversational)
- ~20% at Z4-5 (hard intervals, tempo)
- Minimize sustained Z3 work (tempo runs at lactate threshold) during strength-emphasis blocks

This reflects the polarized training evidence (Stöggl & Sperlich 2014) applied specifically to concurrent athletes, where recovery capacity is already under greater demand.

**Deload frequency:**
Every 3-4 weeks (more frequent than single-sport): reduce both modalities by 30-40%.

## Principle 4b: Minimum Exercise Anchors

For mixed plans, both running and strength anchors apply. Every mixed plan must include across all sessions:
- **Eccentric lower-leg:** Single-leg calf raise with 3s lowering (serves both running economy and strength)
- **Posterior chain:** Nordic curl / sofa curl or RDL (running injury prevention + strength goal)
- **Hip stability:** Clamshell or side-lying abduction (knee health for both modalities)

Place these in the first lower-body strength session of Week 1. Do not omit even if the plan is short.

## Principle 5: Session Planning Logic

**Reasoning process (Claude should apply this, not follow a fixed template):**

1. Identify key sessions for primary goal (e.g., long run, tempo run for runners; heavy compound days for lifters)
2. Place key sessions first in the week — these are non-negotiable
3. Place secondary modality sessions where interference with key sessions is minimal
4. Apply the muscle-group matching rule to every combination
5. Ensure 48h separation between any two sessions that stress the same muscle groups at high intensity
6. Fill remaining days with easy/recovery sessions or rest

**Example reasoning (5 days, endurance primary):**
- Key sessions: Long run + 1 quality run + 2 strength sessions
- Long run: Place on weekend, no strength within 48h before
- Quality run: Mid-week
- Strength 1: After easy run day, 48h before next key run
- Strength 2: After another easy day, 48h before next key run
- The specific days are derived from this logic, not from a fixed Mon/Tue/Wed template

## Principle 6: Cardio Modality Choice

**When adding cardio to a strength-primary program:**
- Cycling preferred over running (less lower-body muscle damage → less interference, can reduce interference by ~30-50% vs. running)
- Rowing: minimal leg interference if technique is sound; targets posterior chain and upper back — good carryover for strength athletes
- Swimming: minimal interference, good active recovery
- Running: highest interference with leg training, schedule carefully

**When adding strength to an endurance-primary program:**
- Strength sessions should not compromise the quality of key endurance sessions
- Low-volume, high-intensity strength (3×5) is more compatible than high-volume moderate-intensity (4×12)

## Principle 7: Red Flags Specific to Mixed Training

| Signal | Reasoning | Action |
|---|---|---|
| Performance declining in both modalities simultaneously | Classic overtraining in concurrent athletes | Reduce total weekly volume by 30%, add rest day |
| Persistent fatigue despite adequate sleep | Cumulative fatigue from dual training stress | Immediate deload, reassess volume |
| Strength gains stalled while running is improving | Interference effect taking hold | Reduce run frequency/volume temporarily, increase strength priority |
| Running economy declining while strength is improving | Possible over-emphasis on hypertrophy (body weight increase) | Shift strength focus to neural (lower reps, heavier load, less volume) |
